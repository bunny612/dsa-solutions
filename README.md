## Folder Guide

- `arrays/` — array manipulation, two pointers, sliding window, prefix sums

- `linked-list/` — traversal, reversal, cycle detection
- `trees/` — BST, traversals, tree DP
- `graphs/` — BFS, DFS, topological sort, union-find
- `dp/` — 1D and 2D dynamic programming
- `greedy/` — greedy algorithms
- `backtracking/` — combinatorial search
- `sorting-searching/` — binary search, custom sort, merge/quick sort
